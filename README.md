This is an app that should allow you to create a small crochet project and enter data one of two ways: an entire row or a set of specific row instructions part by part. It will then allow you to save this. Planned functionality also allows to open up and edit patterns that fall into a format it understands (rows of text). This is specifically for freehanded or projects you're creating without a pattern. I intend to make this more robust as I use it, but this is very specifically something I want to use because I'm annoyed at my notebook's lack of extensibility for crochet pattern making.
